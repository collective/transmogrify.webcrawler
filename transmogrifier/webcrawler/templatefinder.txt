TemplateFinder
==============

This blueprint reads in all teh text of html pages and finds patterns. It will then strip
the reaccuring patterns away to effectively detemplate a site leaving just the html content.


>>> from os.path import dirname
>>> from os.path import abspath
>>> config = """
... [transmogrifier]
... pipeline =
...     webcrawler
...     treeserializer
...     printer
...     
... [webcrawler]
... blueprint = transmogrify.webcrawler.webcrawler
... site_url  = file:///%s/test_staticsite/
...
... [treeserializer]
... blueprint = transmogrify.webcrawler.templatefinder
...
... [printer]
... blueprint = collective.transmogrifier.sections.tests.pprinter
...
... """ % abspath(dirname(__file__)).replace('\\','/')
